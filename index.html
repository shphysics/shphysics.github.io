Research

I am interested in applying computer science and physics in the fields of material science and neuroscience, specifically in using physics-based machine-learning models to improve problem-solving efficiency.

At the Brains in Silicon lab, I am working on analyzing the connectomics in the dataset from MICrONS. At the Materials Computation and Theory Group, I worked on using semi-supervised machine learning and learned atomic embeddings to predict the synthesizability of crystalline inorganic materials. For my Honor Thesis project, I am using a physics-based machine-learning approach that is powered by multi-task learning and meta-learning to predict the synthesizability of different types of materials. Representative projects are highlighted.

Expanding the Silicon Brain: Exploring Neurons-Dendrites’ Connectivity in the MICrONS Dataset
William Cai, Saarthak Sarup, Kwabena Boahen
Stanford Bio-X, 2022
poster | code

This project constructed a pipeline to explore the neurons-dendrites’ connectivity in the Cortical mm^3 Dataset from MICrONS to demonstrate that the information carried by spike sequences corresponds to synapse sequences on dendrites. This project is the pioneer project for The Synaptic Organization of Dendrites Project.

Expanding the Spectrum of Synthesizable Materials Using Positive-Unlabeled Learning Model with Group-Based Sampling Method
William Cai, Evan Antoniuk, Evan Reed
Stanford Materials Science and Engineering, 2020
poster | code

This project first constructed an automated pipeline to extract desired data from the Materials Project Database and the Inorganic Crystal Structure Database. Then the project used Sherlock HPC Cluster to run semi-supervised learning algorithms on the extracted data to predict the synthesizability of novel 2D inorganic materials. Finally, the project investigated the effectiveness of the Group-Based Sampling Method (developed in this project) in addressing the class imbalance problem in the dataset: there were more unlabeled data (synthesizability of the material is unknown) than positive data(synthesizable).
