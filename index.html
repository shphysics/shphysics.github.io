Research

I am interested in applying computer science and physics in the fields of material science and neuroscience, specifically in using physics-based machine-learning models to improve problem-solving efficiency.

At the Brains in Silicon lab, I am working on analyzing the connectomics in the dataset from MICrONS. For my Honor Thesis project, I am using a physics-based machine-learning approach that is powered by multi-task learning and meta-learning to predict the synthesizability of different types of materials. Representative projects are highlighted.

A Physics-based meta-learning augmented Machine Learning Approach to Expand the Spectrum of Synthesizable Materials with Multi-task Learning Technique
William Cai, Austin Sendek
Stanford Engineering Physics Honors Program, 2023 (in progress)
project description

This project will utilize multi-task learning and meta-learning, which are machine learning algorithms that can learn how to learn across a collection of related tasks and datasets and then make more robust predictions, to predict synthesizability for novel materials from different databases. By building on the previous pipeline and using the hyperparameter tuning techniques, this project will incorporate physics-based reasoning into meta-learning augmented machine learning algorithms to learn from multiple material science datasets (e.g. The Open Quantum Materials Database) beyond Materials Project Database and the Inorganic Crystal Structure Database.

CITRUSAFE: A Novel Approach to Combatting the Grapefruit-Drug Effect
Allison Argueta, Jay An, Maya Czeneszew, Joshua Buchi, William Cai, Aaron Choi, Nathan Kim, Andre Yeung,
international Genetically Engineered Machine (iGEM), 2021
website | simulation | code

The project first created a method of blocking the furanocoumarins from interacting with the CYP enzymes in order to address the grapefruit-drug interaction. Then the project determined the theoretical yield rate of CYP6B1 protein in genetically engineered Brewer's yeast. Additionally, the project involved codon optimizations in Brewer's yeast for CYP6B1 from Black swallowtail and P450 Reductase from Housefly.

Expanding the Spectrum of Synthesizable Materials Using Positive-Unlabeled Learning Model with Group-Based Sampling Method
William Cai, Evan Antoniuk, Evan Reed
Stanford Materials Science and Engineering, 2020
poster | code

This project first constructed an automated pipeline to extract desired data from the Materials Project Database and the Inorganic Crystal Structure Database. Then the project used Sherlock HPC Cluster to run semi-supervised learning algorithms on the extracted data to predict the synthesizability of novel 2D inorganic materials. Finally, the project investigated the effectiveness of the Group-Based Sampling Method (developed in this project) in addressing the class imbalance problem in the dataset: there were more unlabeled data (synthesizability of the material is unknown) than positive data(synthesizable).
